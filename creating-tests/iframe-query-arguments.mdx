---
title: "IFrame Query Arguments"
description: "Append query parameters to iframe URLs in AutoProctor's Advanced Settings to customize embedded content."
---

When using the **IFrame/Other** quiz provider, you can append query parameters to your embedded quiz URL through AutoProctor's **Advanced Settings**. This lets you customize how the embedded content is displayed without modifying the original URL.

## How It Works

If your original URL is:
```
www.website.com
```

And you want to add parameters to make it:
```
www.website.com?key1=val1&key2=val2
```

Enter the following in the **Query Arguments** field under Advanced Settings:
```
key1=val1&key2=val2
```

AutoProctor automatically constructs the complete URL by appending your parameters.

## Common Examples

| Parameter | Purpose |
|---|---|
| `hl=en` | Render a Google Form in English |
| `hl=fr` | Render a Google Form in French |
| `embedded=true` | Force embedded mode on certain platforms |

<Note>
Do not include the `?` prefix — AutoProctor adds it automatically when constructing the URL.
</Note>
