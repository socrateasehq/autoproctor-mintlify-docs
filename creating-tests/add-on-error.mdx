## The Problem

The AutoProctor Add-On displays only a **"Help"** option instead of the expected **"Settings"** or **"Results"** options.

This is a known issue that Google Add-Ons face frequently.

## Solution

1. Install the latest version of [Google Chrome](https://www.google.com/intl/en_in/chrome/)
2. Open the Google Form in **Incognito Mode**

If the issue persists, email **hello@autoproctor.co** with a screenshot of the error.
