If the AutoProctor Google Forms add-on displays only a **"Help"** option instead of the expected **"Settings"** or **"Results"** options, you are experiencing a known Google Add-On issue. This happens frequently with Google Workspace add-ons and can usually be resolved by updating Chrome or clearing cached permissions.

{/* TODO: Add screenshot of the add-on showing only the Help option */}

## How to Fix It

<Steps>
  <Step title="Update Google Chrome">
    Install the latest version of [Google Chrome](https://www.google.com/intl/en_in/chrome/) to ensure compatibility with Google Workspace add-ons.
  </Step>
  <Step title="Open in Incognito Mode">
    Open the Google Form in an **Incognito window** (press `Ctrl+Shift+N` on Windows or `Cmd+Shift+N` on Mac). Incognito mode clears cached permissions that often cause this issue.

    {/* TODO: Add screenshot of the add-on working correctly in Incognito mode */}
  </Step>
  <Step title="Reinstall the Add-On (If Needed)">
    If the issue persists, uninstall and reinstall the AutoProctor add-on from the [Google Workspace Marketplace](https://workspace.google.com/marketplace/).

    {/* TODO: Add screenshot of the AutoProctor add-on in the Google Workspace Marketplace */}
  </Step>
</Steps>

<Note>
If none of the steps above resolve the issue, [contact us](/support/contact-us) at **hello@autoproctor.co** with a screenshot of the error and we will get back to you.
</Note>
