title: "AutoProctor Add-On Error"
description: "Fix the issue where the AutoProctor Google Forms add-on only shows 'Help' instead of 'Settings' or 'Results'."
---

If the AutoProctor Google Forms Add-On displays only a **"Help"** option instead of the expected **"Settings"** or **"Results"** options, this is a known Google Add-On issue that can be resolved with a few steps.

## How to Fix It

<Steps>
  <Step title="Update Google Chrome">
    Install the latest version of [Google Chrome](https://www.google.com/intl/en_in/chrome/) to ensure compatibility.
  </Step>
  <Step title="Open in Incognito Mode">
    Open the Google Form in an **Incognito window** (press `Ctrl+Shift+N` on Windows or `Cmd+Shift+N` on Mac). This clears cached permissions that may cause the issue.
  </Step>
  <Step title="Reinstall the Add-On">
    If the issue persists, uninstall and reinstall the AutoProctor Add-On from the Google Workspace Marketplace.
  </Step>
</Steps>

<Note>
If none of the steps above resolve the issue, email **hello@autoproctor.co** with a screenshot of the error.
</Note>
