## Archiving Tests

If you archive a test, it no longer shows up on your Dashboard. This is useful for hiding demo or outdated tests while keeping them accessible.

<Frame>
  <img src="/images/creating-tests/archive-test.png" alt="Archive Test option in settings" />
</Frame>

### How to Archive

1. Go to your **Dashboard**
2. Open the test's **Settings**
3. Click **Archive Test**

### Restoring Archived Tests

Visit the **Archived Tests** section in the left sidebar. From there you can **Restore** a test (returns it to the dashboard) or **Delete** it permanently.

## Deleting Tests

Deletion permanently removes tests from your account, but only applies to tests that have already been archived.

<Note>
Deleting a test will **not** delete the user's answers or proctoring records. That data persists even after removal.
</Note>

### Important Notes

- Deleted tests may reappear if the original quiz provider (like Google Forms) is accessed again
- In multi-collaborator scenarios, deletion only affects the account that initiated it — other collaborators retain access
