Every AutoProctor test begins with an instructions page that explains how proctoring works, which features are active, and what candidates should expect. The page content adapts automatically based on your test configuration.

<Frame>
  <img src="/images/getting-started/instructions-page.png" alt="AutoProctor instructions page shown to candidates before a test" />
</Frame>

## Enabling the Instructions Page

The instructions page is **automatically active on all tests**. Candidates see it immediately after clicking the test link. This feature cannot be disabled.

## Adding Custom Instructions

You can include custom instructions (maximum 200 words) via the **Advanced Settings** section of your test configuration.

<Frame>
  <img src="/images/getting-started/additional-instructions-settings.png" alt="Additional instructions field in AutoProctor Advanced Settings" />
</Frame>

<Note>
Adding custom instructions requires an **Elite plan** subscription.
</Note>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Can I gather additional information from candidates?">
    Yes — use the **Gather Additional Details** feature in Advanced Settings. The form appears on the instructions page and must be completed before the test starts.
  </Accordion>

  <Accordion title="Can I preview the instructions page?">
    Yes — click your own test link to see exactly what candidates will see. This does not consume a credit.
  </Accordion>

  <Accordion title="Does viewing the instructions page use a credit?">
    No — credits are only deducted when a candidate successfully starts the test, not when they view the instructions page.
  </Accordion>

  <Accordion title="Can I remove sections from the instructions page?">
    No — the default sections are fixed. You can only add additional instructions, not remove existing ones.
  </Accordion>
</AccordionGroup>
