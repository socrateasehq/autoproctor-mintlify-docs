The instructions page explains how AutoProctor operates, which proctoring features are active, and what the test-taking experience will entail. Candidates can access a demo test to verify device and browser compatibility.

<Frame>
  <img src="/images/getting-started/instructions-page.png" alt="AutoProctor instructions page for candidates" />
</Frame>

The page content adapts automatically based on your test's configuration settings.

## Enabling the Instructions Page

The instructions page is **automatically active on all tests**. Candidates see it immediately after clicking the test link. This feature cannot be disabled.

## Adding Additional Instructions

Test administrators can include custom instructions (maximum 200 words) via the **Advanced Settings** section.

<Frame>
  <img src="/images/getting-started/additional-instructions-settings.png" alt="Additional instructions settings in AutoProctor" />
</Frame>

<Note>
Adding custom instructions requires an **Elite plan** subscription.
</Note>

## Frequently Asked Questions

**Can I gather additional information from candidates?**
Yes — use the "Gather Additional Details" feature, which appears on the instructions page and must be completed before the test starts.

**Can I preview the instructions page?**
Yes — click your own test link to see the candidate experience.

**Does viewing the instructions page use a credit?**
No — credits are only deducted when a candidate successfully starts the test.

**Can I remove sections from the instructions page?**
No — you can only add additional instructions, not remove existing sections.
