---
title: "What Devices Are Compatible with AutoProctor?"
description: "Supported browsers, operating systems, and minimum versions for AutoProctor's proctoring."
---

AutoProctor's proctoring requires specific browser and device features. For video proctoring, the system needs camera access and browser permissions.

## Supported Browsers and Devices

| Operating System | Browser | Minimum Version |
|---|---|---|
| Windows | Chrome | 82 |
| Windows | Firefox | 78 |
| Windows | Edge | 88 |
| Linux | Chrome | 82 |
| macOS | Safari | 12 |
| macOS | Chrome | 82 |
| macOS | Firefox | 78 |
| Android | Chrome | 82 |
| iOS (iPad/iPhone) | Safari | 14.1 |
| iOS (iPad/iPhone) OS > 16.1 | Chrome | 82 |

<Warning>
Any browser/OS combination **not listed** in the table above is not supported for test attempts.
</Warning>
