AutoProctor uses modern browser technologies for real-time AI monitoring. Your browser and device must meet certain minimum requirements for proctoring to function correctly.

## Supported Browsers and Devices

| Operating System | Browser | Minimum Version |
|---|---|---|
| Windows | Chrome | 82 |
| Windows | Firefox | 78 |
| Windows | Edge | 88 |
| Linux | Chrome | 82 |
| macOS | Safari | 12 |
| macOS | Chrome | 82 |
| macOS | Firefox | 78 |
| Android | Chrome | 82 |
| iOS (iPad/iPhone) | Safari | 14.1 |
| iOS (iPad/iPhone) OS 16.1+ | Chrome | 82 |

<Warning>
Any browser and OS combination **not listed** in the table above is not supported. Candidates using unsupported configurations may experience loading issues or test failures.
</Warning>

<Note>
For the best experience, use a device that is less than 3 years old. AutoProctor runs AI algorithms on the candidate's device, which requires adequate processing power.
</Note>

## Verify Your Device

Before your actual test, take the demo test to confirm your device and browser are compatible:
