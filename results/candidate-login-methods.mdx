## Why login is required

Candidates must log in to AutoProctor when they attempt a test. They cannot attempt the test without logging in. The system requires authentication to differentiate between multiple candidates accessing the same test link from different devices.

## Available login methods

AutoProctor offers three authentication options:

1. **Sign in with Google**
2. **Sign in with Microsoft**
3. **Sign in with Email**

![Login methods](/images/results/login-methods.png)

## Login methods by test type

The available login methods depend on the type of test you are administering.

| Test Type | Available Login Methods |
|---|---|
| Socratease Quizzes | Google, Microsoft, Email |
| Google Forms | Google only |
| Microsoft Forms | Microsoft only |
| IFrame Tests | Google, Microsoft, Email |

<Warning>
If the test type is a Google Form or Microsoft Form, only the corresponding sign-in method is available. Google Forms require Google sign-in, and Microsoft Forms require Microsoft sign-in.
</Warning>
