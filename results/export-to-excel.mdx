## Viewing results

Once students complete and submit a test, results display on the AutoProctor dashboard. The results table shows:

- The test taker's name
- Email address
- Test start and submit time
- Trust scores (if proctoring is enabled)
- Quiz scores (for Socratease Quizzes)

![Results table on dashboard](/images/results/export-results-table.png)

## Exporting to Excel

You can export this table to an Excel file for records purposes. This is useful for:

- Sharing results with colleagues
- Maintaining audit records
- Storing documentation for your institution

The resulting spreadsheet contains a summary of all AutoProctor test-taker results. You can view a [sample exported spreadsheet](https://docs.google.com/spreadsheets/d/1lvkt7n7ZkOushCFYd0ZrTv5YgBAex4CJV78LG88mLx4/edit#gid=0) to see the format.
