## Browser and Device Requirements

| Device | Browser | Minimum Version |
|---|---|---|
| iPhone / iPad | Safari | 13+ |
| Windows / Android / Mac | Google Chrome | 83+ |

- Update your browser to the latest version if you are on an older release
- Your device should ideally be less than 3 years old
- AutoProctor runs AI algorithms on your camera and audio feeds, which requires adequate processing power

<Note>
If your device struggles to run the test, you may need to switch to a different (newer) device. Check [Device Compatibility](/getting-started/device-compatibility) for the full list of supported configurations.
</Note>
