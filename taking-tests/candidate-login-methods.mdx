title: "Candidate Login Methods"
description: "Available login options for candidates taking tests on AutoProctor."
---

Candidates must log in before attempting any test on AutoProctor. This is required to differentiate between submissions when multiple candidates use the same test link.

<Frame>
  <img src="/images/taking-tests/login-methods.png" alt="AutoProctor login options" />
</Frame>

## Available Login Methods

- **Sign in with Google**
- **Sign in with Microsoft**
- **Sign in with Email**

## Login Methods by Test Type

The available sign-in methods depend on the quiz provider:

| Test Type | Available Sign-in Methods |
|---|---|
| Socratease Quizzes | Google, Microsoft, Email |
| Google Forms | Google only |
| Microsoft Forms | Microsoft only |
| IFrame Tests | Google, Microsoft, Email |

<Note>
Google Forms tests only allow Google sign-in, and Microsoft Forms tests only allow Microsoft sign-in. Socratease and IFrame tests support all three methods.
</Note>
