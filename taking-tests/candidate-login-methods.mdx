Candidates must log in before attempting any test on AutoProctor. Login is required to differentiate between submissions when multiple candidates use the same test link.

<Frame>
  <img src="/images/taking-tests/login-methods.png" alt="AutoProctor login options showing Google, Microsoft, and Email sign-in" />
</Frame>

## Available Login Methods

- **Sign in with Google** — uses the candidate's Google account
- **Sign in with Microsoft** — uses the candidate's Microsoft account
- **Sign in with Email** — candidate enters their email address directly

## Login Methods by Quiz Provider

The available sign-in methods depend on which quiz provider you use:

| Quiz Provider | Available Sign-in Methods |
|---|---|
| Socratease Quizzes | Google, Microsoft, Email |
| Google Forms | Google only |
| Microsoft Forms | Microsoft only |
| IFrame Tests | Google, Microsoft, Email |

<Note>
Google Forms tests require Google sign-in because AutoProctor needs to pass the candidate's Google identity to the form. The same applies to Microsoft Forms with Microsoft sign-in. Socratease and IFrame tests have no such restriction.
</Note>
