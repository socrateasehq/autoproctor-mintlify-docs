Socratease quizzes are simpler to take on AutoProctor compared to Google Forms or Microsoft Forms tests. There is only **one submit button**, so there is no risk of losing your answers by clicking buttons in the wrong order.

## Try the Demo Test First

Try the demo before your actual test to experience the interface and confirm your device and browser work correctly.

<Steps>
  <Step title="Open the demo test link">
    Visit [the AutoProctor timed Socratease demo test](https://www.autoproctor.co/tests/start-test/cRoAvQmBaU) on the device and browser you plan to use for your actual test.
  </Step>
  <Step title="Complete the demo">
    Follow the on-screen prompts to verify that the timer and quiz interface load correctly on your device.
  </Step>
  <Step title="Troubleshoot if needed">
    If the demo does not load, try updating your browser to the latest version. If the issue persists, see [Device Compatibility](/getting-started/device-compatibility) for the full list of supported configurations.
  </Step>
</Steps>

<Note>
Timed tests do not require camera or microphone access, but a modern browser is still needed for the timer and quiz interface to function properly.
</Note>

## Browser and Device Requirements

| Device | Browser | Minimum Version |
|---|---|---|
| iPhone / iPad | Safari | 13+ |
| Windows / Android / Mac | Google Chrome | 83+ |

- Update your browser to the latest version if you are running an older release.
- Your device should ideally be less than 3 years old.
- While timed tests do not use camera or microphone proctoring, the quiz interface still requires a modern browser with up-to-date web technologies.
