## Browser and Device Requirements

| Device | Browser | Minimum Version |
|---|---|---|
| iPhone / iPad | Safari | 13+ |
| Windows / Android / Mac | Google Chrome | 83+ |

- Update your browser to the latest version if you are on an older release
- Your device should ideally be less than 3 years old

<Note>
Timed tests do not require camera or microphone access, but a modern browser is still needed for the timer and quiz interface.
</Note>
