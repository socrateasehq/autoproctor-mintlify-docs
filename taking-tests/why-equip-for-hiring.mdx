## Different Types of Tests

Equip supports diverse skill assessments including:
- **Coding skills** — with syntax highlighting and code editors
- **Listening and speaking skills** — voice-based assessments
- **Design evaluations** — beyond traditional text-based questions

## Anti-Cheating Features

### Timer per Question
Each question has its own time limit, making it harder for candidates to look up answers externally.

### Negative Points
Wrong answers deduct points, discouraging random guessing.

### No Copy-Paste
Most online cheating happens via Googling the question and copying the answer. Equip disables copy-paste to prevent this.

### Session Recordings
Captures candidate screen activity including cursor movement, typing, and clicks for playback review.

## Excel Import

You can transfer previously created question banks into Equip using the Excel import feature.
