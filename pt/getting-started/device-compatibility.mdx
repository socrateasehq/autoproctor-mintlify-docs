---
title: "Compatibilidade de dispositivos"
description: "Verifique quais navegadores, sistemas operacionais e dispositivos são compatíveis para realizar testes supervisionados no AutoProctor."
---

O AutoProctor utiliza tecnologias modernas do navegador para monitoramento em tempo real com inteligência artificial. Seu navegador e dispositivo devem atender a certos requisitos mínimos para que a supervisão funcione corretamente.

## Navegadores e dispositivos compatíveis

| Sistema operacional | Navegador | Versão mínima |
|---|---|---|
| Windows | Chrome | 82 |
| Windows | Firefox | 78 |
| Windows | Edge | 88 |
| Linux | Chrome | 82 |
| macOS | Safari | 12 |
| macOS | Chrome | 82 |
| macOS | Firefox | 78 |
| Android | Chrome | 82 |
| iOS (iPad/iPhone) | Safari | 14.1 |
| iOS (iPad/iPhone) OS 16.1+ | Chrome | 82 |

<Warning>
Qualquer combinação de navegador e sistema operacional **não incluída** na tabela acima não é compatível. Candidatos que utilizem configurações não compatíveis podem ter problemas de carregamento ou falhas no teste.
</Warning>

<Note>
Para uma melhor experiência, utilize um dispositivo com menos de 3 anos de uso. O AutoProctor executa algoritmos de inteligência artificial no dispositivo do candidato, o que requer capacidade de processamento adequada.
</Note>

## Verifique seu dispositivo

Antes do seu teste real, realize o teste de demonstração para confirmar que seu dispositivo e navegador são compatíveis:
