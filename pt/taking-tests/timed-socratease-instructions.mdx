Os questionários Socratease são mais simples de realizar no AutoProctor comparados com testes do Google Forms ou Microsoft Forms. Existe apenas **um botão de envio**, então não há risco de perder suas respostas clicando nos botões na ordem errada.

## Faça o Teste de Demonstração Primeiro

Faça a demonstração antes do seu teste real para conhecer a interface e confirmar que seu dispositivo e navegador funcionam corretamente.

<Steps>
  <Step title="Abra o link do teste de demonstração">
    Acesse [o teste de demonstração cronometrado do Socratease](https://www.autoproctor.co/tests/start-test/cRoAvQmBaU) no dispositivo e navegador que você planeja usar para o seu teste real.
  </Step>
  <Step title="Complete a demonstração">
    Siga as instruções na tela para verificar que o temporizador e a interface do questionário carregam corretamente no seu dispositivo.
  </Step>
  <Step title="Solucione problemas se necessário">
    Se a demonstração não carregar, tente atualizar seu navegador para a versão mais recente. Se o problema persistir, consulte [Compatibilidade de Dispositivos](/getting-started/device-compatibility) para a lista completa de configurações compatíveis.
  </Step>
</Steps>

<Note>
Testes cronometrados não requerem acesso à câmera ou microfone, mas um navegador moderno ainda é necessário para que o temporizador e a interface do questionário funcionem corretamente.
</Note>

## Requisitos de Navegador e Dispositivo

| Dispositivo | Navegador | Versão Mínima |
|---|---|---|
| iPhone / iPad | Safari | 13+ |
| Windows / Android / Mac | Google Chrome | 83+ |

- Atualize seu navegador para a versão mais recente se estiver usando uma versão antiga.
- Seu dispositivo deve ter idealmente menos de 3 anos de uso.
- Embora testes cronometrados não usem supervisão por câmera ou microfone, a interface do questionário ainda requer um navegador moderno com tecnologias web atualizadas.
