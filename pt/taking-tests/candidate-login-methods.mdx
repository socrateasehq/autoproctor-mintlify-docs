Os candidatos devem fazer login antes de realizar qualquer teste no AutoProctor. O login é obrigatório para que o AutoProctor possa distinguir entre envios individuais quando múltiplos candidatos usam o mesmo link de teste.

## Métodos de Login Disponíveis

O AutoProctor oferece três opções de login para candidatos:

- **Login com Google** — usa a conta Google do candidato
- **Login com Microsoft** — usa a conta Microsoft do candidato
- **Login com E-mail** — o candidato insere seu endereço de e-mail diretamente (sem necessidade de conta de terceiros)

## Métodos de Login por Provedor de Questionários

Os métodos de login disponíveis dependem do provedor de questionários utilizado:

| Provedor de Questionários | Métodos de Login Disponíveis |
|---|---|
| Socratease Quizzes | Google, Microsoft, E-mail |
| Google Forms | Apenas Google |
| Microsoft Forms | Apenas Microsoft |
| Testes IFrame | Google, Microsoft, E-mail |

<Note>
Testes do Google Forms requerem login com Google porque o AutoProctor precisa passar a identidade Google do candidato para o formulário. O mesmo se aplica ao Microsoft Forms com login Microsoft. Testes do Socratease e IFrame não têm essa restrição, então todos os três métodos de login estão disponíveis.
</Note>

<Note>
Para restringir quais candidatos podem fazer login no seu teste, consulte [Restringir por E-mail](/settings/restricting-by-email). Você pode limitar o acesso a endereços de e-mail ou domínios de e-mail específicos.
</Note>
