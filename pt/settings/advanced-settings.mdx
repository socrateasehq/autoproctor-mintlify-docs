As Configurações Avançadas permitem que você controle como os candidatos fazem login, quem pode acessar o teste e como os resultados são compartilhados. Essas opções estão localizadas na parte inferior da página de Configurações do Teste.

## Provedores de Login

Escolha quais métodos de autenticação os candidatos podem usar para acessar o teste:

| Provedor | Descrição | Observações |
|---|---|---|
| **Google** | Login com uma conta Google | Obrigatório para testes com Google Forms |
| **Microsoft** | Login com uma conta Microsoft | Obrigatório para testes com Microsoft Forms |
| **E-mail** | Login com qualquer endereço de e-mail | Disponível para testes Socratease e Iframe |

Para testes com Google Forms e Microsoft Forms, o provedor de login é fixo de acordo com a plataforma do questionário. Para Questionários Socratease e Testes Iframe, você pode ativar um ou mais provedores.

Para mais detalhes sobre como cada método de login funciona, consulte [Métodos de Login do Candidato](/taking-tests/candidate-login-methods).

## Restrições de Login

Restrinja quais candidatos podem realizar seu teste com base no endereço de e-mail ou domínio.

- **Restringir por E-mail** — Permita apenas endereços de e-mail específicos para acessar o teste
- **Restringir por Domínio** — Permita apenas candidatos com endereços de e-mail de domínios específicos (por exemplo, @universidade.edu)

Para instruções detalhadas, consulte [Restringir por E-mail](/settings/restricting-by-email). Se você deseja convidar um conjunto específico de candidatos, consulte [Convidar Candidatos por E-mail](/settings/inviting-candidates-via-email).

## Altura do IFrame

Quando seu teste está incorporado como um iframe dentro do AutoProctor, esta configuração permite ajustar a altura (em pixels) desse iframe para corresponder ao conteúdo do seu teste. O valor padrão é 1764px — aumente-o se os candidatos precisarem rolar desnecessariamente, ou diminua-o para remover espaço em branco extra.

## Integração com Google Sheets

Exporte automaticamente os resultados do teste para uma planilha do Google Sheets inserindo a URL da planilha neste campo. Para questionários Socratease, tanto os resultados de supervisão quanto as notas do questionário são exportados. Para questionários que não são do Socratease, apenas os resultados de supervisão são incluídos.

Para instruções de configuração, consulte [Integração com Google Sheets](/results/google-sheets-integration).

## Colaboradores

Adicione colegas como colaboradores no seu teste para que possam editar configurações e visualizar resultados. Você pode definir o Administrador Principal do teste clicando no botão **Manage**.

Para mais detalhes, consulte [Adicionar Colaboradores](/creating-tests/adding-collaborators).

## Recursos Premium e Elite

### Ao Concluir o Teste (Premium)

Controla o que os candidatos veem após enviar o teste. Por padrão, todos os usuários veem a página de conclusão padrão do AutoProctor. Usuários Premium e superiores podem personalizar isso com duas opções adicionais:

| Opção | Descrição |
|---|---|
| **Mostrar mensagem padrão** | Exibe a página de conclusão padrão do AutoProctor (disponível em todos os planos) |
| **Mostrar mensagem personalizada** | Exibe sua própria mensagem HTML para os candidatos (por exemplo, "Obrigado por completar a avaliação!") |
| **Redirecionar para URL** | Envia os candidatos para uma página externa após o envio (por exemplo, o site da sua empresa ou um formulário de feedback) |

### URL Única por Candidato (Elite)

Por padrão, todos os candidatos compartilham um único link de teste, que pode ser encaminhado para destinatários não previstos. Ativar esse recurso gera um link único para cada candidato, garantindo que apenas as pessoas com quem você compartilhou os links possam realizar o teste.

<Note>
URL única por candidato é um recurso Elite. Consulte [Recursos Elite](/pricing/elite-features) para mais detalhes.
</Note>

### Coletar Informações Adicionais (Premium)

Colete informações personalizadas dos candidatos antes do início do teste — como número de telefone, nome da instituição ou qualquer pergunta personalizada. Esses campos aparecem na página de instruções e os dados coletados ficam disponíveis nos resultados e nas exportações para Excel.

<Note>
Você pode adicionar até 10 campos de informações personalizadas por teste.
</Note>

### Instruções Adicionais (Elite)

Adicione instruções personalizadas à página de instruções do candidato. Elas são exibidas para os candidatos antes de iniciarem o teste e podem ser usadas para fornecer diretrizes específicas, regras ou contexto para o seu teste.

<Note>
Instruções Adicionais é um recurso Elite. Consulte a [Página de Instruções para Candidatos](/taking-tests/instructions-page-for-candidates) para mais informações sobre o que os candidatos veem antes de iniciar.
</Note>
