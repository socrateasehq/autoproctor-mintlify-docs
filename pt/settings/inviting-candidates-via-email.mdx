title: "Convidar Candidatos por E-mail"

---

Os links de convite únicos permitem que você gere uma URL de teste distinta para cada candidato, vinculada ao endereço de e-mail dele. Quando o candidato abre o link, o AutoProctor envia um código de verificação de uso único para confirmar a identidade antes de conceder acesso ao teste.

<Note>
Este recurso está disponível apenas no **Plano Elite** e funciona exclusivamente com **Socratease Quizzes** e **Testes IFrame**. Não é compatível com Google Forms nem com Microsoft Forms, pois esses exigem login específico da plataforma. Consulte [Pagamentos e Créditos](/pricing/payments-and-credits) para detalhes do plano.
</Note>

## Como a Verificação Funciona

Quando o candidato clica no seu link único, o AutoProctor envia um código de verificação de uso único para o endereço de e-mail associado àquele link. O candidato deve inserir esse código para acessar o teste. Isso garante que apenas o destinatário pretendido possa usar cada link.

## Limitações Importantes

| Limitação | Detalhes |
|---|---|
| URL padrão desativada | Ao ativar este recurso, a URL compartilhada padrão do teste para de funcionar. Apenas as URLs únicas por candidato concedem acesso. |
| Sem revogação | Você não pode invalidar convites enviados após serem gerados. |
| Sem teste gratuito | Este recurso não pode ser testado antes de adquirir o Plano Elite. |
| Limite por lote | Você pode fazer upload de no máximo 1.000 endereços de e-mail por vez. |

<Warning>
Após ativar URLs únicos, o link comum do teste para de funcionar imediatamente. Certifique-se de que você está pronto para distribuir os links individuais antes de ativar este recurso.
</Warning>

## Serviço de Envio de E-mail em Massa

Para campanhas em grande escala que excedam 5.000 candidatos, o AutoProctor oferece um serviço de distribuição de e-mail a $100 por cada 5.000 mensagens. Isso requer saldo suficiente na conta e o envio do seu arquivo CSV e conteúdo do e-mail para a equipe do AutoProctor. [Entre em contato conosco](/support/contact-us) para mais detalhes.
