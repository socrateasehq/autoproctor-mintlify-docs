As etiquetas são rótulos de metadados que você adiciona às perguntas do questionário para organizar, classificar e filtrar os resultados dos testes. Elas ajudam você a analisar o desempenho dos candidatos por disciplina, competência, dificuldade ou qualquer outra categoria que você definir.

<Note>
As etiquetas são **visíveis para os candidatos** durante o teste. Mantenha os nomes das etiquetas profissionais e informativos (por exemplo, `math`, `Python`, `section-A`).
</Note>

## O Que Você Pode Fazer com Etiquetas?

As etiquetas funcionam tanto em **Questionários** quanto em **Bancos de Perguntas**. Depois de etiquetar suas perguntas, o relatório do questionário oferece duas capacidades poderosas:

- **Filtrar** perguntas por etiquetas específicas para ver apenas os resultados dessas perguntas
- **Agrupar** perguntas por etiquetas para visualização e análise organizadas

Exemplos de etiquetas incluem:
- Categorização por disciplina: `inglês`, `história`, `matemática`
- Identificação de competências: `Python`, `JavaScript`, `SQL`
- Designação de dificuldade: `fácil`, `médio`, `difícil`

## Como Usar Etiquetas

<Steps>
  <Step title="Crie ou Abra um Questionário Socratease">
    Crie um novo Questionário Socratease no AutoProctor ou abra um existente.
  </Step>
  <Step title="Adicione Etiquetas às Perguntas">
    Após adicionar uma pergunta, clique no **menu de três pontos** no canto superior direito da pergunta. Insira o nome da etiqueta no campo de texto que diz "Add tag & Press Enter" e pressione **Enter** para adicioná-la. Você pode adicionar múltiplas etiquetas a uma única pergunta.

    {/* TODO: Add screenshot of the three-dot menu with tag input field */}
  </Step>
  <Step title="Salve o Questionário">
    Salve seu questionário após adicionar todas as etiquetas. As etiquetas persistem com a pergunta e são transferidas para os resultados.
  </Step>
  <Step title="Veja Resultados Agrupados por Etiquetas">
    Depois que os candidatos enviarem o teste, abra os resultados do teste. Use o menu suspenso **"Group By"** e selecione **Tags** para ver as perguntas organizadas por suas respectivas etiquetas.

    {/* TODO: Add screenshot of the Group By dropdown with Tags selected */}
  </Step>
  <Step title="Filtre Resultados por Etiqueta">
    Use o menu suspenso **"Filter by Tag"** para selecionar etiquetas individuais e exibir apenas as perguntas com essas etiquetas específicas. Isso permite analisar o desempenho em tópicos ou competências específicos.

    {/* TODO: Add screenshot of the Filter by Tag dropdown in action */}
  </Step>
</Steps>
