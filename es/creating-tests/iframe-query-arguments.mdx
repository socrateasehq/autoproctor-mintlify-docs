---
title: "Argumentos de consulta del iframe"
description: "Agregue parámetros de consulta a las URL de iframe en la Configuración avanzada de AutoProctor para personalizar el contenido integrado."
---

Al utilizar el proveedor de cuestionarios **IFrame/Other**, puede agregar parámetros de consulta a la URL de su cuestionario integrado a través de la **Configuración avanzada** de AutoProctor. Esto le permite personalizar cómo se muestra el contenido integrado sin modificar la URL original.

## Cómo funciona

Si su URL original es:
```
www.website.com
```

Y desea agregar parámetros para que sea:
```
www.website.com?key1=val1&key2=val2
```

Ingrese lo siguiente en el campo **Query Arguments** en la Configuración avanzada:
```
key1=val1&key2=val2
```

AutoProctor construye automáticamente la URL completa agregando sus parámetros.

## Ejemplos comunes

| Parámetro | Propósito |
|---|---|
| `hl=en` | Mostrar un Google Form en inglés |
| `hl=fr` | Mostrar un Google Form en francés |
| `embedded=true` | Forzar el modo integrado en ciertas plataformas |

<Note>
No incluya el prefijo `?` — AutoProctor lo agrega automáticamente al construir la URL.
</Note>
