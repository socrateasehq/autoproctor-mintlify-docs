---
title: "Compatibilidad de dispositivos"
description: "Verifique qué navegadores, sistemas operativos y dispositivos son compatibles para realizar pruebas supervisadas en AutoProctor."
---

AutoProctor utiliza tecnologías modernas del navegador para el monitoreo en tiempo real con inteligencia artificial. Su navegador y dispositivo deben cumplir ciertos requisitos mínimos para que la supervisión funcione correctamente.

## Navegadores y dispositivos compatibles

| Sistema operativo | Navegador | Versión mínima |
|---|---|---|
| Windows | Chrome | 82 |
| Windows | Firefox | 78 |
| Windows | Edge | 88 |
| Linux | Chrome | 82 |
| macOS | Safari | 12 |
| macOS | Chrome | 82 |
| macOS | Firefox | 78 |
| Android | Chrome | 82 |
| iOS (iPad/iPhone) | Safari | 14.1 |
| iOS (iPad/iPhone) OS 16.1+ | Chrome | 82 |

<Warning>
Cualquier combinación de navegador y sistema operativo **no incluida** en la tabla anterior no es compatible. Los candidatos que utilicen configuraciones no compatibles pueden experimentar problemas de carga o fallos en la prueba.
</Warning>

<Note>
Para una mejor experiencia, utilice un dispositivo con menos de 3 años de antigüedad. AutoProctor ejecuta algoritmos de inteligencia artificial en el dispositivo del candidato, lo cual requiere una capacidad de procesamiento adecuada.
</Note>

## Verifique su dispositivo

Antes de su prueba real, realice la prueba de demostración para confirmar que su dispositivo y navegador son compatibles:

[Realizar la prueba de demostración](https://www.autoproctor.co/tests/start-test/GFICZZZA)
