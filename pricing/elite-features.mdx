## Elite-Only Features

As you can see on the [Pricing Page](https://www.autoproctor.co/pricing), AutoProctor has 3 different tiers for its subscriptions: **Standard**, **Premium**, and **Elite**. The Elite Plan has the most features. It includes all the features present in the Standard and Premium Plans, plus the following.

### 1. API/SDK Access

If you already have your own quizzing or assessment platform and would like to enable proctoring on it, you can use our JavaScript SDK.

### 2. Cloning Quizzes

You can **duplicate Socratease Quizzes in a single click**. If you are creating multiple quizzes that are slightly different from one another, this is a very handy feature.

<Note>
The following features (3-5) are relevant if you are using Socratease Quizzes.
</Note>

### 3. Question Banks

With Socratease Quizzes, Question Banks help you:

1. **Select a random subset of questions of different difficulty levels.** For example, 5 easy and 2 hard MCQs from a pool of 100 MCQs.
2. **Combine questions from different pools.** For example, 3 Physics and 2 Chemistry MCQs from 2 separate pools of 100 MCQs each.

### 4. More Question Types

Socratease Quizzes have many different question types. The following are available only on the Elite Plan:

<CardGroup cols={3}>
  <Card title="Voice Input">
    Create a question which requires the candidate to record themselves speaking the answer.
  </Card>
  <Card title="Autograded Text">
    Specify the correct answer when creating the question. Candidates type their submission in an input box. If their submission matches the correct answer exactly, they get full points. Otherwise, 0 points.
  </Card>
  <Card title="Answer Any">
    Create 5 questions and allow the candidate to answer any 3 of them (numbers are configurable).
  </Card>
</CardGroup>

### 5. Import From Excel

Using our Excel template, you can **bulk upload questions** onto a Socratease Quiz.
