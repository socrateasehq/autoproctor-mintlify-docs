Socratease Quizzes feature 13 different question types available across different subscription tiers. All question types on a lower plan are available on higher plans -- meaning Premium subscribers access all basic features plus advanced ones, while Elite users get complete access.

| Question Type | Free | Premium | Elite |
|---|:---:|:---:|:---:|
| MCQ | Yes | Yes | Yes |
| MCA | Yes | Yes | Yes |
| Short Text | Yes | Yes | Yes |
| Single Line Input | Yes | Yes | Yes |
| Coding Editor | -- | Yes | Yes |
| Cloze | -- | Yes | Yes |
| Match | -- | Yes | Yes |
| Categorize | -- | Yes | Yes |
| Comprehension | -- | Yes | Yes |
| Document | -- | Yes | Yes |
| Answer Any | -- | -- | Yes |
| Voice Input | -- | -- | Yes |
| Autograded Text | -- | -- | Yes |
