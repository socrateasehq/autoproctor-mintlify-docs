Socratease offers 13 distinct question formats to assess candidates across different skill areas. These range from traditional multiple-choice to advanced formats like voice recording, drag-and-drop matching, and auto-graded text responses.

<Note>
Not all question types are available on every plan. See [Question Type Availability](/socratease/question-type-availability) for details on which types are included in your plan.
</Note>

## Available on All Plans

### 1. MCQ (Multiple Choice Question)

A question with only one correct answer. Candidates select a single option from the available choices.

<Frame>
  <img src="/images/socratease/mcq.png" alt="MCQ question type showing a single-answer multiple choice question with radio buttons" />
</Frame>

### 2. MCA (Multiple Correct Answers)

A question with multiple correct answers. Candidates can select more than one option using checkboxes.

<Frame>
  <img src="/images/socratease/mca.png" alt="MCA question type showing a multiple correct answers question with checkboxes" />
</Frame>

### 3. Short Text

An essay-type question where you can set minimum and maximum character length for the response. Use this for open-ended answers that require more than a single line.

<Frame>
  <img src="/images/socratease/short-text.png" alt="Short text question type showing a multi-line text area with character limits" />
</Frame>

### 4. Single-line Input

A compact text input field for shorter answers. You can set validations to accept only specific formats like emails, numbers, or dates.

<Frame>
  <img src="/images/socratease/single-line-input.png" alt="Single-line input question type showing a text field with validation options" />
</Frame>

## Available on Premium and Elite Plans

<Note>
The following question types require a **Premium** or **Elite** plan. See [Question Type Availability](/socratease/question-type-availability) for the full breakdown.
</Note>

### 5. Coding Editor

Candidates write code with syntax highlighting and autocomplete. You choose the programming language. The code cannot be executed -- this question type evaluates the candidate's ability to write code, not to run it.

<Frame>
  <img src="/images/socratease/coding-editor.png" alt="Coding editor question type showing a code editor with syntax highlighting" />
</Frame>

### 6. Cloze (Fill-in-the-Blanks)

A fill-in-the-blank format where candidates see the available options (including wrong ones) and drag and drop them into the correct blanks within the text.

<Frame>
  <img src="/images/socratease/cloze.png" alt="Cloze question type showing fill-in-the-blank with drag-and-drop options" />
</Frame>

### 7. Match

A matching exercise where candidates drag and drop items to align related pairs across two columns.

<Frame>
  <img src="/images/socratease/match.png" alt="Match question type showing two columns with drag-and-drop matching" />
</Frame>

### 8. Categorize

Candidates classify a set of items into the correct buckets or categories by dragging and dropping them.

<Frame>
  <img src="/images/socratease/categorize.png" alt="Categorize question type showing items being sorted into category buckets" />
</Frame>

### 9. Comprehension

A nested question format where you write a passage of text and ask multiple questions based on that passage. All sub-questions appear alongside the comprehension text.

<Frame>
  <img src="/images/socratease/comprehension.png" alt="Comprehension question type showing a text passage with nested sub-questions" />
</Frame>

### 10. Document

Upload a PDF document and ask multiple questions based on its content. Candidates view the document alongside the questions.

<Frame>
  <img src="/images/socratease/document.png" alt="Document question type showing a PDF viewer with associated questions" />
</Frame>

## Available on Elite Plan Only

<Note>
The following question types are exclusive to the **Elite** plan. See [Elite Features](/pricing/elite-features) for more about Elite plan capabilities.
</Note>

### 11. Answer Any

You create a set of questions (for example, 5) and let candidates choose a subset to answer (for example, any 3 out of the 5). This gives candidates flexibility while maintaining assessment rigor.

<Frame>
  <img src="/images/socratease/answer-any.png" alt="Answer Any question type showing a set of questions with selection flexibility" />
</Frame>

### 12. Voice Input

You type a piece of text and candidates record themselves speaking that text. This is useful for language assessments, pronunciation evaluations, and oral communication testing.

<Frame>
  <img src="/images/socratease/voice-input.png" alt="Voice input question type showing a text prompt with audio recording controls" />
</Frame>

### 13. Autograded Text

You type the question text and the expected answer. The candidate's response is automatically marked as correct if it matches the expected answer exactly. This enables instant grading without manual review.

<Frame>
  <img src="/images/socratease/autograded-text.png" alt="Autograded text question type showing a text input with automatic answer matching" />
</Frame>
