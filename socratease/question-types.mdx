Socratease offers 13 distinct question formats to assess candidates and students across different skill areas. These range from traditional multiple-choice to advanced formats like voice recording and auto-graded text responses.

<Note>
Not all question types are available on every plan. See [Question Type Availability](/socratease/question-type-availability) for details on which types are included in your plan.
</Note>

## Available on All Plans

### 1. MCQ (Multiple Choice Question)

A question with only one correct answer. Candidates select a single option from the available choices.

<Frame>
  <img src="/images/socratease/mcq.png" alt="MCQ question type" />
</Frame>

### 2. MCA (Multiple Correct Answers)

A question with multiple correct answers. Candidates can select more than one option.

<Frame>
  <img src="/images/socratease/mca.png" alt="MCA question type" />
</Frame>

### 3. Short Text

Essay-type question where you can set minimum and maximum length for the response.

<Frame>
  <img src="/images/socratease/short-text.png" alt="Short text question type" />
</Frame>

### 4. Single-line Input

Like Short Text, but for smaller inputs. You can set validations, like accepting only emails, numbers, or dates.

<Frame>
  <img src="/images/socratease/single-line-input.png" alt="Single-line input question type" />
</Frame>

## Available on Premium and Elite Plans

### 5. Coding Editor

Candidates can write code with syntax highlighting, autocomplete, and more. You can choose the programming language. The code cannot be executed -- it is meant for evaluating the candidate's ability to write code.

<Frame>
  <img src="/images/socratease/coding-editor.png" alt="Coding editor question type" />
</Frame>

### 6. Cloze (Fill-in-the-Blanks)

Fill-in-the-blanks, but the candidates can see the different options (including wrong ones) and can drag and drop them into the correct blanks.

<Frame>
  <img src="/images/socratease/cloze.png" alt="Cloze question type" />
</Frame>

### 7. Match

Match related items by dragging and dropping items across two columns.

<Frame>
  <img src="/images/socratease/match.png" alt="Match question type" />
</Frame>

### 8. Categorize

Given a set of items, the candidate must classify them into the right bucket or category.

<Frame>
  <img src="/images/socratease/categorize.png" alt="Categorize question type" />
</Frame>

### 9. Comprehension

Nest questions within each other. Write a piece of text and ask multiple questions based on that piece.

<Frame>
  <img src="/images/socratease/comprehension.png" alt="Comprehension question type" />
</Frame>

### 10. Document

Upload a document as a PDF and ask multiple questions based on that document.

<Frame>
  <img src="/images/socratease/document.png" alt="Document question type" />
</Frame>

## Available on Elite Plan Only

### 11. Answer Any

Create (say) 5 questions and let the candidates choose any (say) 3 out of the 5 questions to answer.

<Frame>
  <img src="/images/socratease/answer-any.png" alt="Answer Any question type" />
</Frame>

### 12. Voice Input

Type a piece of text and ask the candidates to record themselves speaking that text. Useful for language assessments and pronunciation evaluations.

<Frame>
  <img src="/images/socratease/voice-input.png" alt="Voice input question type" />
</Frame>

### 13. Autograded Text

You type in the question text and the expected answer. The candidate's answer is automatically marked as correct if their submission matches the expected answer exactly.

<Frame>
  <img src="/images/socratease/autograded-text.png" alt="Autograded text question type" />
</Frame>
