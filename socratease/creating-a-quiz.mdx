Socratease is AutoProctor's built-in quizzing platform, designed to be as easy as Google Forms but purpose-built for online examinations. You can create quizzes, distribute them to candidates, and review results all from a single dashboard.

Key highlights include:
- A **single [submit button](/taking-tests/submit-button)** -- no confusion between submitting the quiz and the proctored test
- **Integrated creation and results** viewing within one dashboard
- Support for **13 question formats** alongside advanced capabilities like Excel imports and question banks
